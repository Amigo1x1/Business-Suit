Технологія Ambilight
Динамічна фонова під світка екрану телевізора чи монітора комп’ютера (аналог Philips Ambilight).
Яка працює під управлінням Arduino, на комп’ютері працює програма Ambibox.
Arduino керує адресною світлодіодною стрічкою. На чіпах WS2812.
В схему добавлений фоторезистор для адаптивного налаштування яскравості стрічки в залежності від інтенсивності освітлення в приміщенні.

• Дуже дешевий аналог Ambilight для любого монітора/телевізора підключеного до комп’ютера.
• Роздільність саморобної фонової підсвітки набагато вища, чим пропонують навіть самі дорогі моделі від Philips.
• Зручна програма Ambibox для налаштування і персоналізації фонової підсвітки.

Технологія Ambilight розробляється для 2-х типів пристроїв

Для настільних ПК Для телевізорів
